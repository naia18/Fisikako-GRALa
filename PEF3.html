2. Emaitzen alderaketa¶

Aurreko programetan \(n\_part\) browndar partikulako sistema bat simulatu dugu. Horretarako, higidura-ekuazioetatik abiatu, eta horiek partikula guztientzat ebatziz, posizio eta abiadura guztiak denboran eraberritzean oinarritzen zen kodea idatzi da ( Euler-Maruyama metodoan oinarrituta). Lortutako traiektoriak \(R_t\) array batean gordetzen ziren, eta gero, hauek simulatu genituen.

Lortutako emaitza guztiak begi-bistaz posibleak baziruditen ere, hauek balio teorikoekin alderatzea komenigarria da. Baina noski, ezin dut simulazioa bera balio teorikoekin alderatu; ejekutatzen den aldi bakoitzeko ausazko traiektoriak bueltatuko dizkit (horixe baita higidura browndarraren beraren koxka); beraz, nola ziurta genezake kodearekin lortzen diren posizio/abiadura/ausazko indarrak egokiak diren?

Gure emaitza esperimentalak (hau da, simulaziotik lortutakoak), emaitza teorikoekin aldera ditzakegu.

Higidura browndarrari buruzko teoria, batez ere, L. Reichl-en A Modern Course in Statistical Physics liburuan irakurri dut.

  1. Auto-korrelazio funtzioa (dimentsio baterako):

\[\langle\langle v(t_2)v(t_1)\rangle_\xi\rangle_T = \frac{k_B T}{m}e^{-(\gamma/m)|t_2-t_1|}\]

Eta simulazioko 3 dimentsiotako sistemara zabalduz, eta kontsideratuz \(t_1=0\) dela:

\[\phi_v(t)=\langle \vec{V}(t)\cdot \vec{V}(0)\rangle = \langle V_x(t)V_x(0)\rangle + \langle V_y(t)V_y(0)\rangle + \langle V_z(t)V_z(0)\rangle = \frac{3k_BT}{m}\cdot e^{-(\gamma/m)|t|}\]

  1. Batez besteko desplazamendu koadratikoa:

\[ \langle \langle (r(t)-r_0)^2 \rangle_\xi\rangle_T = \frac{6 k_B T}{\gamma}\left[t-\frac{m}{\gamma}(1-e^{-(\gamma/m)t})\right]\]

  1. Parikularen abiaduraren dentsitate espektrala:

\[S_{v,v}=\int_{-\infty}^\infty d\tau e^{-i\omega\tau}\langle v_\alpha(t_1+\tau)v_\alpha(t_1) \rangle_\zeta = \frac{2k_BT\gamma}{m^2\omega^2+\gamma^2}\]

Goiko adierazpenean, \(\alpha\) ikurrak edozein norabide adierazten du. Simulazioko 3 dimentsiotako sistemara zabalduz:

\[S_v(\omega)=\frac{6k_BT}{m}\frac{\gamma_c}{\omega^2+\gamma_c^2}\]

Non \(\gamma_c=\gamma/m\) aurreko auto-korrelazio funtzioko indargetze konstantea den (decay constant).

Beraz, gure kodea erabilita abiaduraren auto-korrelazio funtzioa eta dentsitate espektrala kalkulatu ditzakegu eta gero balio teoriko hauekin alderatu.

2.1. Problemaren ebazpena¶

# LIBRERIEN INPORTAZIOA
%matplotlib inline
import numpy as np
import matplotlib.pyplot as plt
import matplotlib as mpl
from matplotlib import rc
import scipy.integrate as scp
from numpy import fft    # dentsitate espektrala kalkulatzeko (aurreragoko atalean)
plt.style.use('fivethirtyeight')

# PARAMETROEN HASIERATZEA
dim = 3 
dt = 0.05
gamma = 1.0
m = 1.0
kBT = 1.0
d_est = np.sqrt(2*kBT*gamma*dt)
n_part = 1000       # sistema osatuko duten part. kopurua
n_pausu = 1024      # simulaziorako erabiliko ditugun denbora-pausuak
np.random.seed(0)   # hazia = 0-ko ausazko zenbakien sortzailea hasieratu

# ALDAGAIEN HASIERATZEA
R = np.zeros([n_part,dim])
V = np.zeros([n_part,dim])
W = np.zeros([n_part,dim])
R_t = np.zeros([n_pausu,n_part,dim])
V_t = np.zeros([n_pausu,n_part,dim])
W_t = np.zeros([n_pausu,n_part,dim])
denbora = np.zeros([n_pausu])

# HIGIDURA-EKUAZIOAREN EBAZPENA
for i in range(n_pausu):
    W = d_est*np.random.randn(n_part,dim)
    R, V = R + V*dt, V*(1-gamma/m*dt)+W/m
    R_t[i] = R  # aldiune bakoitzari dagokion partikula guztien hiru ardatzetako posizioak gorde
    V_t[i] = V  # aldiune bakoitzari dagokion partikula guztien hiru ardatzetako abiadurak gorde 
    W_t[i] = W  # aldiune bakoitzari dagokion partikula guztien hiru ardatzetako indarrak gorde
    denbora[i] = i*dt

Inline plotting-a erabiltzeko: (https://docs.wradlib.org/en/stable/notebooks/python/mplintro.html).

Pythoneko komando bat da, oso erabilgarria, ingurunearen “propietateak” (environment settings) kontrolatzen dituena. Honela, kodigoa exekutatzean notebook-ean bertan exekutatuko zaizkigu plot-ak. Komando hau sartuko ez bagenu, beste leiho bat irekiko litzaiguke.

2.2. Auto-korrelazioa¶

def auto_korrelazio(v):
    kor = np.correlate(v,v,mode="full")  # Bi 1D-ko sekuentzien korrelazioa (numPy manualetik) 
    return kor[len(v)-1:]/len(v)

kor = np.zeros([n_pausu])

# Baina ohartu hiru dimentsioak hartu behar ditugula kontuan!!

for i in range(n_part):
    for l in range(dim):
        kor = kor + auto_korrelazio(V_t[:,i,l])   #  i. partikularen l dimentsioari dagokion aldiune guztietako abiaduren autokorrelazioa
kor = kor/n_part

mpl.rc('xtick', labelsize=20) 
mpl.rc('ytick', labelsize=20) 
plt.rc('text', usetex=True)
plt.rc('font', family='serif')

fig, ax = plt.subplots(figsize=(8,8),dpi=300)
ax.plot(denbora, dim*kBT/m*np.exp(-gamma/m*denbora),lw=4, label=r'$\frac{3k_BT}{m}e^{-(\gamma/m)|t|}$ (Teo)', alpha=0.6)
ax.plot(denbora, kor,lw=4,label=r'$\langle \mathbf{v}(t)\cdot\mathbf{v}(0) \rangle_{\mathrm{esp}}$ (Esp)',alpha=0.6)
ax.set_xlabel(r"$t$", fontsize=24)
ax.set_ylabel(r"$\phi_V(t)$",fontsize=24)
ax.set_xlim(-0.2,8)
ax.legend(fontsize=22)
plt.show()
fig.savefig('auto_kor.png')

Auto-korrelazioa kalkulatzeko erabili dugun funtzioak numpy libreriako correlate funtzioa erabiltzen du. Zuzenean libreriako dokumentazioan irakurri dudanetik:

This function computes the correlation as generally defined in signal processing texts:

\[c_{av}[k] = \sum_n a[n+k] * conj(v[n])\]

Gure kasuan \(v\) bektore beraren auto-korrelazio funtzioa kalkulatu nahi dugunez,

\[C_{v}[k]= \sum_n v[n+k]*v[n]\]

Behar dugun bataz-besteko egokia lortzeko, behin partikula guztien 3 dimentsiotako auto-korrelazioa kalkulatuta, hauxe partikula kopuru totalarekin zatitu behar dugu.

Balio esperimentalei dagokion kurbak zentzua du: denborak aurrera egin ahala, \(t\rightarrow\infty\), browndar partikularen hasierako abiadurari buruzko informazioa esponentzialki txikituz dihoa. Teoriarekin bateragarria da.

2.3. Batez besteko desplazamendu koadratikoa¶

# BBDK-AREN KALKULUA
bbdk = np.zeros([n_pausu])             # pausu bakoitzeko bbdk
for i in range(n_pausu):
    for part in range(n_part):
        bbdk[i] = bbdk[i] + np.linalg.norm(R_t[i,part,:])**2
    bbdk[i] = bbdk[i]/n_part

# DIFUSIO KONSTANTEAREN KALKULUA
T = denbora[-1]
Difusio_ktea_1 = scp.trapz(bbdk,dx=dt)/(3*T**2)   # Difusio ktea kalkulatu trapezioaren erregelarekin integratuz
Difusio_ktea_2 = scp.simps(bbdk,dx=dt)/(3*T**2)   # Difusio ktea kalkulatu simpsonen erregelarekin integratuz
D_teoriko = kBT/gamma

print("Difusio konstantearen balio teorikoa: "+str(D_teoriko))
print("Difusio konstantea trapezioaren metodoaz integratuz: "+str(Difusio_ktea_1))
print("Difusio konstantea simpsonen metodoaz integratuz: "+str(Difusio_ktea_2))

# DATUEN IRUDIKAPENA
mpl.rc('xtick', labelsize=20) 
mpl.rc('ytick', labelsize=20) 
plt.rc('text', usetex=True)
plt.rc('font', family='serif')

fig, ax = plt.subplots(figsize=(8,8),dpi=300)
ax.plot(denbora,6*kBT/gamma*(denbora-m/gamma*(1-np.exp(-denbora))), lw=3, label=r"$\frac{6k_BT}{\gamma}\left[t-\frac{m}{\gamma}(1-e^{-(\gamma/m)t})\right]$ (Teo)",alpha=0.7)
ax.plot(denbora, bbdk, lw=3,label=r"$\langle\langle (\mathbf{r}(t)-\mathbf{r}(0))^2 \rangle_\xi\rangle_T$ (Esp)",alpha=0.7)
ax.set_xlabel(r"$t$", fontsize=24)
ax.set_ylabel(r"BBDK",fontsize=24)
ax.legend(fontsize=20)
plt.show()
fig.savefig('bbdk.png')

2.4. Partikulen abiaduraren potentzia dentsitate espektrala¶

def pds(v,dt):
    vw = fft.fft(v)*dt  # V(w), vw(0)-n 0 maiztasuneko elementuarekin
    return np.abs(vw[:n_pausu//2])**2/(n_pausu*dt)  # S_v, w>0 den blioetarako
Sw = np.zeros([n_pausu//2])
for i in range(n_part):
    for l in range(dim):
        Sw = Sw + pds(V_t[:,i,l],dt)  # 
Sw = Sw/n_part

plt.rc('text', usetex=True)
plt.rc('font', family='serif')

fig, ax = plt.subplots(figsize=(8,8),dpi=300)
gamma = gamma/m
omega = fft.fftfreq(n_pausu,d=dt)[:n_pausu//2]*2.0*np.pi
ax.plot(omega,(6*kBT/m)*gamma/(omega**2 + gamma**2),lw=3,label=r'$\frac{6k_BT}{m}\cdot\frac{\gamma_c}{\omega^2+\gamma_c^2}$ (Teo)',alpha=0.6)
ax.plot(omega,Sw,lw=3,label=r'$|\mathbf{V}_{\tau}(\omega)|^2/\tau$ (Esp)',alpha=0.6)
ax.set_xlabel(r"$\omega$",fontsize=24)
ax.set_ylabel(r"$S_{VV}(\omega)$",fontsize=24)
plt.xlim(-0.2,8)
plt.ylim(bottom=-0.2)
ax.legend(fontsize=22)
plt.show()
fig.savefig('Dents_esp.png')

Kasu hontan ere teorian lortutako adierazpena eta emaitza esperimentalak guztik bateragarriak direla lortzen dugu. Beraz, ondoriozta genezake, oro har, kodea zuzena dela.

Dena dela, oraindik badira interesgarriak liratekeen adierazpen grafikoak. Adibidez, partikula guztien posizioak denboraren funtzioan adierazi genitzake. Sarritan egiten da hau browndar partikulen higidurak deskribatzeko, emaitza bisuala polita baita (eta esanguratsua).

2.5. Partikula ororen \(R\) posizioak VS \(t\) denbora¶

plt.rc('text', usetex=True)
plt.rc('font', family='serif')
fig, ax = plt.subplots(figsize=(7.5,7.5), dpi=200) # gogoratu inch-etan dagoela
ax.set_xlabel(r"$t$", fontsize=20)
ax.set_ylabel(r"$R_{\alpha}(t)$", fontsize=20)
ax.set_xlim(0,50)
ts = 10
for n in range(n_part):  #partikula guztietarako
    lx, = ax.plot(denbora[::ts],R_t[::ts,n,0],'c',lw=1, alpha=0.1)
    ly, = ax.plot(denbora[::ts],R_t[::ts,n,1],'m',lw=1, alpha=0.1)
    lz, = ax.plot(denbora[::ts],R_t[::ts,n,2],'y',lw=1, alpha=0.1)

leg = ax.legend([lx,ly,lz], [r"$x$",r"$y$",r"$z$"], loc=0, fontsize=14)

for l in leg.get_lines():
    l.set_alpha(1)

plt.show()
fig.savefig('R_VS_t.png')

\(R_x\), \(R_y\) eta \(R_z\) denboraren funtzioan adierazi dira, hortaz, posizio bertikalak \(x\), \(y\) edo \(z\) posizioa zehaztuko digu.

\(t=0\) aldiunean, partikula guztiak daude jatorrian eta denborak aurrera egin ahala, \(t>0\)-n, era estokastikoan mugituz doaz.

Zer atera genezake grafiko hontatik? Hasteko, begi-bistan dago bataz besteko posizioa zero dela edozein aldiunetan, baina bestalde, bariantza handituz doala: erdiko zonalde dentsoa geroz eta zabalagoa da denborak aurrera egiten duen heinean, baina betiere erdian zentratuta dago.
