1. Higidura Browndarraren azterpena jito-indar batekin¶

Orain arte ikusitakoa laburbilduz gero:

  1. Mugimendu Browndarraren 2D eta 3Dko simulazioak burutu dira, independenteak diren (eta hortaz, beraien arteko elkarrekintzarik ikusten ez duten) partikula browndarren kasurako.

  2. Hortaz gain, FuncAnimation libreriaz baliatuz, 3Dtako simulazio animatu bat egitea lortu da.

Bi puntu hauek garatu ditugu, kontsideratuta betiere partikula browndarren erreferentzia sisteman gaudela.

Hala ere, kasu erreal batetarantz hurbiltzen hasteko, demagun ozeanoan askatu dugun tinta beltza osatzen duten partikulen MZko sisteman egon beharrean, prozesua kanpotik behatzen duen txalupa batetako erreferentzia sisteman gaudela (laborategi sistema izango litzatekeena). Demagun ere, tinta beltzezko tanta hau, itsaslaster batean erori dela. Itsaslaster bat ozeanoetako ur mugimendu jarraitua da, ezaugarritzat norabidea, abiadura eta emari jakin bat dituena. Itsaslasterrak, itsas urei eragiten dieten indarrek sortarazten dituzte (Lurraren errotazioak, haizeak, tenperatura nahiz gazitasun desberdintasunek, itsasaldiek…). Itsaslasterraren metafora jartzen dugun era berean jarri genezake ere odolean globulo gorriak higitzen diren adibidea. Globulo gorriak partikula browndarrak lirateke, eta odolaren jarioaren norabidea, jito-indarra.

Noski, itsaslaster edo odol-jario hauexek, indar bat eragingo dute partikula browndarrengan. Indar hau konstantea dela suposatuko dugu, eta hasiera batean, norabide jakin batetan dihoana. Tankera hontako kanpo-indar baten aurrean sistema batek nola erantzuten duten ikustea zeharo baliagarria izan daiteke, sistema horren propietate dinamikoak aztertzerakoan. Langevin-en ekuazioa, kasu honetan, honela geratuko litzateke:

\[\dot{\vec{V}}(t)=-\frac{\gamma}{m}\vec{V}+\frac{\vec{\xi}}{m}+\frac{\vec{F}}{m} \tag{1}\]

non \(\vec{F}=F_0 \hat{i}\) jito-indar konstante bat den (norabidea guk aukeratutakoa), kasu erreal batetan agertuko litzatekeena jariakin batean murgildutako partikula browndarrak aztertzerakoan.

Era iteratiboan \(\boldsymbol{R}(t)\) posizioa eta \(\boldsymbol{V}(t)\) abiadura eguneratzeko, Euler-Maruyamaren metodoaz baliatuko gara oraingoan ere:

\[\begin{split}\boldsymbol{R}_{i+1}=\boldsymbol{R}_i+\boldsymbol{V}_i\cdot\Delta t\\ \boldsymbol{V}_{i+1}=\boldsymbol{V}_i\left(1-\frac{\xi}{m}\cdot dt\right)+\frac{\boldsymbol{W}_i}{m}+\frac{F}{m} \tag{2}\end{split}\]

# ----------- PARAMETROAK ---------------
dim = 3
dt = 0.05
m = 1.0  
gamma = 1.0                       # Marruskadura koefizientea 
kBT = 1.0                         # Tenperatura [K], abiadura \sqrt(3) [m/s] izan daiten
sigma = np.sqrt(2*gamma*kBT*dt)   # Desbideratze estandarra (ausazko indarraren modelizazioaren anplituderako)
n_part = 1000
n_pausu = 1000

# --------- ALDAGAI HASIERATUAK ----------
R_d = np.zeros([n_part,dim])             # 3 osagaien aldiuneko posizioa jito-indarraren presentzian
V_d = np.zeros([n_part,dim])             # 3 osagaien aldiuneko abiadura jito-indarraren presentzian
W_d = np.zeros([n_part,dim])             # 3 osagaien aldiuneko ausazko indarra jito-indarraren presentzian
R_t_d = np.zeros([n_pausu,n_part,dim])   # 3 osagaien posizio guztiak jito-indarraren presentzian
V_t_d = np.zeros([n_pausu,n_part,dim])   # 3 osagaien abiadura guztiak jito-indarraren presentzian
W_t_d = np.zeros([n_pausu,n_part,dim])   # 3 osagaien ausazko indar guztiak jito-indarraren presentzian
denbora = np.zeros([n_pausu])            # Denbora array-a
F = np.zeros([n_part,dim])              
F[:,0] = 1.0                             # x-ardatzeko indar kte bat, jito-indarra
# --------- SIMULAZIOA ----------

np.random.seed(0)                        # hazia hasieratu
for i in range(n_pausu):
    W_d = sigma*np.random.randn(n_part,dim)
    V_d = (1-gamma/m*dt)*V_d+W_d/m+F/m*dt
    R_d = R_d + V_d*dt
    R_t_d[i,:,:]=R_d
    V_t_d[i,:,:]=V_d
    W_t_d[i,:,:]=W_d
    denbora[i] = i*dt

1.2. Emaitza esperimentalak teorikoekin alderatzen¶

1.2.1. Difusio konstantea kalkulatzen¶

Jito-indarraren pean, emaitza esperimentalak teorikoekin alderatzeko, difusio-koefizientea kalkulatzen saiatuko gara, Einstein-en erlazioaz baliatuz.

Langevin-en ekuazioko egoera egonkorra hartuz, \(t\rightarrow \infty\) limitean:

\[\lim_{t\rightarrow\infty}\left\langle\frac{\text{d}\vec{V}}{\text{d}t}\right\rangle=\lim_{t\rightarrow\infty}\left\langle -\frac{\gamma}{m}\vec{V}+\frac{\vec{\xi}}{m}+\frac{\vec{F}}{m} \right\rangle \tag{3}\]

Definizioz, egoera egonkorreko abiadura-aldaketa nulua izan beharko litzateke. Hots, egoera egonkorra erdiestean, abiadura konstantea izan beharko litzateke \(x\) ardatzean, batezbestean, eta gainera, nulua \(y\) eta \(z\) ardatzetan. Bestalde, definizioz ere, badakigu ausazko indarraren batez bestekoa nuloa dela. Eta azkenik, jito-indar konstantea, konstantea denez, honen batez bestekoa konstantea bera izango da:

\[0=\lim_{t\rightarrow\infty}\left\langle -\frac{\gamma}{m}\vec{V}+0+\frac{\vec{F}}{m} \right\rangle\]
\[lim_{t\rightarrow\infty}\langle \vec{V_X}(t)\rangle=\frac{F_0}{\gamma}\quad\Rightarrow\quad \boxed{D=\lim_{t\rightarrow\infty}\langle \vec{V_X}(t)\rangle\frac{k_BT}{F_0}} \tag{4}\]

Izan ere, Einsteinen erlaziotik (Einstein’s kinetic theory) badakigu, ekuazioaren forma orokorrena:

\[D=\mu k_BT\]

Dela, non \(\mu=v_d/F_0\) den, \(v_d\) drift abiadura izanik.

Analisia burutzeko, lehenbizi jito-indarrik pairatzen ez duten partikulen azterpena egingo da: Difusio koefizientea eta batezbesteko desplazamendu koadratikoa (Mean Square Displacement ) esperimentalki kalkulatu eta balio teorikoekin (Reichl liburuko literaturatik ateratakoak) alderatuko dira. Segidan, emaitza berak aztertuko dira jito-indar bat bai pairatzen duten partikula browndarren kasurako.

1.3.2. Init() eta animatu(i) funtzioak¶

def init():
    global R, V, W, F_elk, R_t, V_t, W_t, denbora, F_d
    
    # Objektuak hasieratu
    
    titulu1.set_text(r'')
    titulu2.set_text(r'')
    
    ibilbidea.set_data([],[])
    ibilbidea.set_3d_properties([])
    
    partikulak.set_data([],[])
    partikulak.set_3d_properties([])
    
    # Aldagaiak hasieratu
    
    hasieratu_R()
    
    V[:,:] = 0.0
    W[:,:] = 0.0
    F_elk[:,:] = 0.0
    
    R_t[:,:,:] = 0.0
    R_t[0,:,:] = R[:,:]
    V_t[:,:,:] = 0.0
    W_t[:,:,:] = 0.0
    F_d[:,:]= 0.0
    F_d[:,0] = 1.0 
    
    denbora[:] = 0.0
    
    return partikulak, titulu1, ibilbidea, titulu2


def animatu(i):
    global R, V, W, F_elk, R_t, V_t, denbora, F_d
    
    part_elkarrekintza()
    
    W = W_dest*np.random.randn(n_part,dim)
    V = ((1-gamma/2/m*dt)*V + F_elk*dt/m + W/m + F_d*dt/m)/(1+gamma/2/m*dt)
    R = R + V*dt
    
    R_t[i,:,:] = R
    V_t[i,:,:] = V
    W_t[i,:,:] = W
    
    denbora[i] = i*dt
    
    titulu1.set_text(r"t = "+str(denbora[i])+" s.d.u.")
    titulu2.set_text(r"OZEANOA")
    
    ibilbidea.set_data(mbp(R_t[:i+1,n,0],gelaxka_aldea),mbp(R_t[:i+1,n,1],gelaxka_aldea))
    ibilbidea.set_3d_properties(mbp(R_t[:i+1,n,2],gelaxka_aldea))
    
    partikulak.set_data(mbp(R[:,0],gelaxka_aldea),mbp(R[:,1],gelaxka_aldea))
    partikulak.set_3d_properties(mbp(R[:,2],gelaxka_aldea))
    
    return partikulak, titulu1, ibilbidea, titulu2

1.3.3. Funtzio gehigarriak¶

# GAINEZARMENA EKIDITZEKO, PARTIKULEN HASIERAKO POSIZIOA KONFIGURATU
def hasieratu_R():
    global R, V, W, R_t, V_t, W_t, F_elk, denbora
    
    # Partikula denentzat errepikatu
    for n in range(n_part):
        gainezarmena = True   
        while gainezarmena:        # errepikatu gainezarmena TRUE denerako
            R[n,:]=np.random.rand(dim)*gelaxka
            gainezarmena = False   
            for k in range(n):     # aztertu gainezarritako posizioak
                rij = distantzia(R[n,:],R[k,:],gelaxka) 
                rij2 = np.linalg.norm(rij)**2     # rij**2
                if rij2 < (0.9*sig)**2:           # Ikusi ea distantzia guk ezarritako limitea baino txikiagoa den 
                    gainezarmena = True    # ezarri gainezarmena TRUE bezala
                    
def distantzia(r1,r2,gelaxka_aldea):
    return r2-r1-np.around((r2-r1)/gelaxka_aldea)*gelaxka_aldea

# INDAR INTER-PARTIKULARRA KALKULATU
def fij(rij2,rij): # Lennard-Jones
    f_ij = -48*eps*((rij2/sig**2)**(-6))/rij2*rij + 24*eps*((rij2/sig**2)**(-3)/rij2*rij)
    return f_ij
def part_elkarrekintza():
    global F_elk
    F_elk[:,:] = 0.0
    for n in range(n_part):  # errepikatu partikula guztietarako
        rij = distantzia(R[n,:], R, gelaxka)  # rij= R_i-R_j
        rij2 = np.linalg.norm(rij, axis=1)**2  # distantzia karratura
        auzokide = (rij2 < (2.0*sig)**2)  # BAKARRIK HARTZEN DITUGULA KONTUAN 2*\SIGMA DIST BAINO GUTXIAGORA DAUDEN PARTIKULAK
        auzokide[n] = False  # i=j ez izan kontuan partikula bera
        F_elk[n,:] = np.sum(fij(rij2[auzokide, newaxis], rij[auzokide,:]), axis=0)

# MUGALDE BALDINTZA PERIODIKOAK      
def mbp(r, gelaxka_aldea):  # posizio guztiei MBP-ak inplementatu
    return np.fmod(r+gelaxka_aldea,gelaxka_aldea)
